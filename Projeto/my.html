MY BEAUTIFUL DARK TWISTED FANTASY

O Album MY BEAUTIFUL DARK TWISTED FANTASY é para Kanye West como "Ultimato" é para a franquia Marvel no cinema: um final grandioso para amarrar toda uma história e partir para um novo começo. Para o público, foi sempre esperar pelo próximo arrasa-quarteirão; para West, foi o início de uma megalomania que, aos poucos, vai vencendo o rapper.
